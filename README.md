## Uso
##### Crear archivo config.json
```
{
    "apiUrl": "http://192.168.0.36:8080"
}
```
##### Instalar dependencias
```
npm install
```
##### Instalar expo
```
npm install -g expo-cli
```
##### Levantar la app (previamente levantar la api rest)
```
npm start
```
